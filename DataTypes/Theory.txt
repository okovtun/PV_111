Переменная (Variable) - это именованная область памяти,
содержимое которой может изменять в процессе выполнения программы.

						type name;	//синтаксис объявления переменной

type - тип объявляемой переменной. 
Он определяет сколько памяти переменная будет занимать,
какие значения она сможет принимать, и какие операции над ней можно выполнять.

1. Логические типы: bool (Boolean) занимает 1Byte памяти,
   и принимает одно из двух значений true либо false.
   true - правда	1
   false - ложь		0
   cout << true << endl;
   cout << false << endl;
   false - это 0
   true - все что НЕ 0
 
2. Символьные типы: char (Character - символ) занимает 1 Байт памяти, и содержит
   1 единственные символ в кодировке ASCII. Всего ASCII-символов 256;

3. Числовые типы 
	(floating point)
	Целочисленные бывают:
	unsigned - беззнаковые, могут хранить лишь положительные целые числа.
	  signed - знаковые, могут хранить как положительные, так и отрицаетльные целые числа.

	  short
	  long
	  int
	  long long
